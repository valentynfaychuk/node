# The AMA Protocol

A Privacy Layer 1 for Agentics /w MatMul-based Useful Proof of Work

## Protocol Parameters

| Parameter | Value |
|-----------|-------|
| Block Time | 500ms |
| Epoch | 100,000 blocks |
| Max Cap | 1 billion AMA |
| UPoW | MatMul |
| Signature | BLS12-381 |
| Contracts | WASM |
| Proofs | Sorted Merkle Tree |
| Serializer | VecPak |
